这是一个基于 nonebot 的插件，用于实现 VDS DEVELOPER 所支持的fursuit tv 端点。

请先到 https://developer.vds.pub/ 获取 VDS DEVELOPER 密钥，

在env文件里面配置

app_id=你的id
client_secret=你的密钥

xdgop 星电计划 CYXbot 附属插件
